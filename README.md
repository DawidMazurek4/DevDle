# DevDle

DevDle is a guessing game for programming languages inspired by Wordle. The player tries to identify a programming language based on hints about its characteristics.

## About the project

The goal of the game is to guess the correct programming language using information such as its typing system, year of creation, paradigm, memory management, and typical use cases.

The project is split into a backend and a frontend.

## Tech stack

Backend:
- Go
- Gin
- PostgreSQL

Frontend:
- Next.js

## Features

- Guess programming languages based on hints
- Feedback showing how close the guess is
- REST API communication between frontend and backend
- Database of programming languages
